### [lab2](lab2/)
  A short drill about:
  Commands:
  ```sql
      SELECT <column>
      FROM <table>
      WHERE <condition>
  ```
  Conditions:

  ```sql
      IS NULL
      IN BETWEEN (values)
      LIKE <charsequence>
      ORDER BY <column> {ASC|DESC}
  ```
  Clauses:
```sql
      GROUP BY <columns>
      ORDER BY <columns> {ASC|DESC}
```
